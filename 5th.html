4. Secure Hash Calculation

For secure hash use HMAC-SHA256 hashing scheme. This scheme requires secretKey and plain text, which is going to be hashed. The resultant hash will be in hex format.

The secretKey will be shared by ePay, and the plain text for hashing scheme will be:
