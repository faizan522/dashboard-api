Payment API

1. Payment API (JazzCash)

URL: https://{BASE_URL}/epay/jazzcash

Type: POST

Content Type: application/json

Header: Authorization > Bearer {TOKEN}

Request:

Parameter Mandatory Data Type Description
merchantTransactionId Yes String Merchant’s system unique Id
billRefNo Yes String Merchant’s unique bill id
(It can be same as merchantTransactionId)
amount Yes Decimal Amount of transaction considers up to 4 decimal places
consumerNo Yes String User’s e-wallet no.
(Format 03xxxxxxxxx)
payeeName Yes String User’s name
purchaseItem Yes String Item which is being sold
payeeCNIC Yes String User’s CNIC without dashes
(Format 422xxxxxxxxxx)
payeeEmail No String User’s email address
currency No String Default PKR

Response:

{
    "result": {
        "transactionId": 1234
    },
    "responseCode": "100",
    "responseMessage": "Transaction processed successfully",
    "timeStamp": "2024-01-01T23:56:17.7968012+05:00"
}
